Modul: Filemount-Übersicht (Filemounts View)
============================================

Alphabetische Auflistung der Filemounts
---------------------------------------

Mit einem Klick auf die Spaltenüberschrift "Bezeichnung" erhalten Sie eine aufsteigend sortierte Liste der Filemounts.
Mit einem zweiten Klick auf die Spaltenüberschrift erhalten Sie eine absteigend sortierte Liste der Filemounts.

Neuen Filemount anlegen
-----------------------

Wählen Sie oben in der Auswahlbox "Einen neuen Filemount anlegen". Es wird ein Formular angezeigt, mit dem Sie einen
neuen Filemount anlegen können.

.. image:: Images/manual_html_6feebec6.png
   :class: with-shadow


Sortierung nach Kriterien
-------------------------

Die Filemount-Liste kann nach verschiedenen Kriterien sortiert werden. Die Liste wird nach der aktuellen
Spaltenüberschrift sortiert, die Sortierung ist durch eine Markierung erkenntlich. Durch Anklicken einer
Spaltenüberschrift können Sie die Sortierung verändern.
